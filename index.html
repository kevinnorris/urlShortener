URL Shortener Microservice

Add /new/ and a url as a parameter to this url, and this service will respond with a JSON object that contains the the original passed url and a shortened url.

When you visit the shortened url you will be redirected to the original url.

Example creations

https://urlm.herokuapp.com/new/https://www.google.com https://urlm.herokuapp.com/new/http://foo.com:80

Example creation output

{ "original_url": "http://foo.com:80", "short_url": "https://urlm.herokuapp.com/5f" }

Usage

https://urlm.herokuapp.com/5f will redirect to http://foo.com:80
