URL Shortener Microservice

Add a url as a parameter to this url and this service will respond with a JSON object that contains the the original passed url and a shortened url.

When you visit the shortened url you will be redirected to the original url

Example creation

https://thissite.com/ourl

Example creation output

{ "original_url": "ourl", "short_url": "surl" }

Usage

surl will redirect to ourl
